💡 How Your Application APIs Connect to Databases

1. Registration API (signup.php)
  • Database: MySQL
  • Action: Stores user credentials (username, email, hashed password)
  • Security: Uses prepared statements to prevent SQL injection
2. Login API (login.php)
  • Databases: MySQL + Redis
  • Actions:
    • MySQL: Verify user credentials
    • Redis: Store session token (expires in 1 hour)
3. Profile API (get_profile.php & update_profile.php)
  • Databases: Redis + MongoDB
  • Actions:
    • Redis: Validate session token
    • MongoDB: Retrieve/update user profile data
4. Logout API (logout.php)
  • Database: Redis
  • Action: Delete session token
