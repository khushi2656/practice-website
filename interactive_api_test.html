🔌 Interactive API Database Tester

Test your database connections step-by-step using APIs

1 Test Database Connections via API

First, let's verify all databases are connected using the API endpoint.

Endpoint: GET /php/api_test.php?action=all

2 Register User via API

Register a new user account using the signup API.

Endpoint: POST /php/signup.php
Database: MySQL

3 Login via API

Authenticate and get session token from the API.

Endpoint: POST /php/login.php
Databases: MySQL (verify user) + Redis (store session)

4 Get Profile via API

Retrieve user profile data from MongoDB using session validation.

Endpoint: POST /php/get_profile.php
Databases: Redis (validate session) + MongoDB (get profile)

5 Update Profile via API

Update user profile data in MongoDB.

Endpoint: POST /php/update_profile.php
Databases: Redis (validate session) + MongoDB (update profile)

6 Logout via API

Clear session from Redis.

Endpoint: POST /php/logout.php
Database: Redis (delete session)
