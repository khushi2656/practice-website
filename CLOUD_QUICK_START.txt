┌───────────────────────────────────────────────────────────────────────┐
│ STEP 1: MONGODB ATLAS (Cloud Database - FREE)                        │
└───────────────────────────────────────────────────────────────────────┘

    🌐 Go to: https://www.mongodb.com/cloud/atlas/register
    
    ✅ Sign up → Choose FREE M0 Cluster (512MB)
    ✅ Create Database User: admin / Admin@123
    ✅ Network Access: Allow 0.0.0.0/0 (Access from Anywhere)
    ✅ Get Connection String:
       mongodb+srv://admin:Admin@123@cluster.xxxxx.mongodb.net/
    
    📝 Create Database: user_profiles_db
    📝 Create Collection: profiles

    ⏱️ Time: 5-10 minutes


┌───────────────────────────────────────────────────────────────────────┐
│ STEP 2: REDIS CLOUD (Session Storage - FREE)                         │
└───────────────────────────────────────────────────────────────────────┘

    🌐 Go to: https://redis.com/try-free/
    
    ✅ Sign up → Create FREE Database (30MB)
    ✅ Provider: AWS
    ✅ Region: Choose closest to you
    ✅ Database name: user-sessions
    
    📝 Copy Connection Details:
       • Endpoint: redis-xxxxx.cloud.redislabs.com:xxxxx
       • Password: xxxxxxxxxxxxxxxx

    ⏱️ Time: 3-5 minutes


┌───────────────────────────────────────────────────────────────────────┐
│ STEP 3: UPDATE CONFIG FILE                                           │
└───────────────────────────────────────────────────────────────────────┘

    📂 Open: php/config.php
    
    🔧 Replace MongoDB URI (around line 35):
       Before: mongodb+srv://admin:Admin@123@user-profiles-cluster.xxxxx...
       After:  [Your actual MongoDB Atlas connection string]
    
    🔧 Replace Redis Host (around line 50):
       Before: redis-12345.c123.us-east-1-1.ec2.cloud.redislabs.com
       After:  [Your actual Redis Cloud endpoint]
    
    🔧 Replace Redis Port (around line 51):
       Before: 12345
       After:  [Your actual Redis Cloud port]
    
    🔧 Replace Redis Password (around line 52):
       Before: your-redis-cloud-password
       After:  [Your actual Redis Cloud password]

    💾 SAVE the file!

    ⏱️ Time: 2 minutes


┌───────────────────────────────────────────────────────────────────────┐
│ STEP 4: TEST LOCALLY (Optional but Recommended)                      │
└───────────────────────────────────────────────────────────────────────┘

    1️⃣ Install XAMPP from: https://www.apachefriends.org/
    2️⃣ Copy project to: C:\xampp\htdocs\practice-website\
    3️⃣ Start Apache in XAMPP Control Panel
    4️⃣ Run database_setup.sql in phpMyAdmin
    5️⃣ Open: http://localhost/practice-website/environment_checker.html
    
    ✅ Check: Should show "Running on: LOCALHOST"
    ✅ Check: MySQL, MongoDB, Redis all connected

    ⏱️ Time: 10-15 minutes


┌───────────────────────────────────────────────────────────────────────┐
│ STEP 5: RAILWAY DEPLOYMENT (Host Your App Online)                    │
└───────────────────────────────────────────────────────────────────────┘

    🌐 Go to: https://railway.app/
    
    ✅ Login with GitHub
    ✅ New Project → Deploy from GitHub repo
    ✅ Select: khushi2656/practice-website
    ✅ Add Service → Database → MySQL
    
    ⚙️ Set Environment Variables (in your web service):
       
       DB_HOST=[From Railway MySQL connection info]
       DB_PORT=[From Railway MySQL connection info]  
       DB_USER=root
       DB_PASS=[From Railway MySQL connection info]
       DB_NAME=railway
       
       MONGO_URI=[Your MongoDB Atlas connection string]
       MONGO_DB=user_profiles_db
       
       REDIS_HOST=[Your Redis Cloud endpoint]
       REDIS_PORT=[Your Redis Cloud port]
       REDIS_PASSWORD=[Your Redis Cloud password]
    
    🗄️ Setup MySQL Database:
       • Click MySQL service → Connect
       • Run: CREATE DATABASE railway;
       • Run: Copy/paste contents of database_setup.sql
    
    🌍 Generate Domain:
       • Click your web service
       • Settings → Generate Domain
       • Get URL: https://your-app.up.railway.app

    ⏱️ Time: 15-20 minutes


┌───────────────────────────────────────────────────────────────────────┐
│ STEP 6: TEST ONLINE                                                  │
└───────────────────────────────────────────────────────────────────────┘

    🌍 Open: https://your-app.up.railway.app/environment_checker.html
    
    ✅ Check: Should show "Running on: ONLINE"
    ✅ Check: All databases connected
    
    🎉 Test the complete flow:
       https://your-app.up.railway.app/interactive_api_test.html

    ⏱️ Time: 2 minutes
